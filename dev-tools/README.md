# Development Tools Directory

This directory contains development utilities, diagnostic scripts, and testing tools for the WhatsApp Gateway project.

## Contents

### Database Tools
- `check-database.js` - Analyze and inspect database records
- `init-database.js` - Initialize SQLite database with tables and sample data
- `final-solution.js` - Troubleshoot and fix database storage issues

### Testing Scripts
- `test-integration.js` - Comprehensive integration testing suite
- `diagnose-774577134.js` - Diagnostic tool for specific phone number issues
- `send-text.js` - Simple script to send test WhatsApp messages

### Configuration Tools
- `check_env_keys.js` - Verify environment variables
- `setup-meta-webhook.js` - Automated Meta webhook setup
- `verify-signed-webhook.js` - Verify webhook signature authenticity

### Other Utilities
- `sms.json` - SMS configuration data
- `test-vonage.json` - Vonage API test configuration

## Usage

These tools are meant for development and debugging purposes only. They should not be used in production environments without proper review.
